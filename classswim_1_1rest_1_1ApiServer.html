Detailed Description

Implements a minimalist REST API for a GossipFailureDetector server. Uses GNU libmicrohttpd as the underlying HTTP daemon.

In the current implementation it is uniquely associated with a reference to the detector server, thus it must not outlive the referenced server's life.

See also
https://www.gnu.org/software/libmicrohttpd/
