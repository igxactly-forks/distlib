◆ split()

std::vector<std::string> utils::split ( const std::string &  values,
const std::string &  sep = ",",
bool  trim_spaces = true,
bool  preserve_empty = true 
)

Splits a sep-separated string into its components part.

For example using the default ',' comma separator, it would return "one,two,three" as a three-element vector {"one", "two", "three"}.

It can optionaly skip empty values (by default, they are returned as empty strings) and remove leading and trailing spaces around the values (the default behavior).

Spaces inside the values (unless space is used as sep) are preserved.

Parameters
valuesthe string containing 0 or more string separated by sep
septhe separator; by defaul the comma (,) but can be any string
trim_spacesif true (the default) spaces around the values are trimmed
preserve_emptyif true (the default) consecutive sep strings will be considered as a single separator (in other words, "one,two,,,,three" will return a three-elements vector
Returns
a vector of string containing the values, possibly empty.
